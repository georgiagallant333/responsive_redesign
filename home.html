Usability Problems

First, it was important to identify the areas in which the current interface provided a challenging user experience

Unclear navigation

It is not apparent how to interact with elements to see more information

Unclear navigation

The "Boarding" heading is a collapsible accordion, however nothing indicates that Boarding is collapsible. The only way to collapse the accordion is to scroll down, and click the "Grooming" accordion below.

Previous Next
Inconsistent highlighting colors

The same color represents selected and non-selected elements.

Inconsistent Highlighting Colors

Information tab is selected while rates and food are not, yet they are all the same highlighted color

Messy Image Integration

Many of the images Bird Camp uses make the website look chaotic and detract from the central information the page is conveying

Messy Image Integration

The background image is a mess of toys of vibrant colors.

Too much text

While this page is a page to convey information, there is too much text to make the content digestible

Too much text

The food tab consists of information already conveyed on the policy page, and until the content is read it is not clear what "Food" under "Boarding" would mean.

Non-intuitive Information Display

Much of the information is conveyed in a haphazard way, that is hard upon first glance, to understand.

Non-intuitive Information Display

The rates page is prime area for a table, but instead is conveyed in a series of small paragraphs

Additionally there were several accessibility related issues
  1. The WAVE reader recognizes all of the navigational elements within the Information page (Boarding, Rates, Food, Grooming) as an unordered list. So majority of the page would be not-navigable by visually impaired users
  2. None of the images have alternative text descriptions so the screen reader cannot describe what they feature
  3. The Information page has no heading elements
  4. The screen reader doesn't convey the need to scroll down, so half the page (the "Grooming" tab) would be overlooked

Accessibility Observations

It was important to check that this redesigned website was, in fact, responsive and accessible. I tested the website under the following conditions and observed:
  1. Varying Window Sizes: With varying window sizes the text wraps, bird images shrink and the navbar collapses (hamburger style)
  2. Varying text size: Because the text wraps, the varying font sizes don't drastically vary the appearance of the browser
  3. Screen Reader: Can accurately read off the elements (largely text) of the page
  4. Different Devices: On mobile and tablet the navbar collapses. Mobile and tablet look identical to a small browser
